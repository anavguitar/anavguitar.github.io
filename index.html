CSS Colors

RGBA

Function Type Value One Value Two Value Three Value Four Example
RGBA() 
Red Channel, 0-255
 
Blue Channel, 0-255
 
Green Channel, 0-255
 
Alpha or Opacity, 0-1 
(use decimals to change opacity)
 
background-color: rgba(200, 0, 0, 1);
#HEX 
Red Channel, R or RR 
Hex num 0-ff (0-255)
 
Green Channel, G or GG 
Hex num 0-ff (0-255)
 
Blue Channel, B or BB 
Hex num 0-ff (0-255)
 
Alpha or Opacity, 0-ff(255)
 
background-color: #00bb00ff;
HSLA() 
Hue, 0-360deg

Saturation, 0-100%
100% completely saturated, 0% gray
 
Lightness, 0-100%
100% is white, 0% is black, 50% is normal
 
Alpha or Opacity, 0-100%
                    		
                    
                        
background-color: hsla(240 100% 50% / 1)

                    		
                

                

                    Color Name Keywords
                    Color name keywords can be used to set color properties
                    Easiest method to color items, but limited when compared to RGBA
                    background-color: purple;
